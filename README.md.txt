 📈 Stock Market Prediction using Machine Learning

This project uses historical stock data to predict the next day's closing price using **Linear Regression** and **Random Forest** models. The dataset is pulled using the `yfinance` library and features are engineered using rolling averages and price change percentages.



 🔍 Features
 📊 Fetches historical data using `yfinance`
 📈 Calculates 50-day & 200-day moving averages
 🤖 Trains Linear Regression & Random Forest models
 🧠 Saves models using `joblib`
 📉 Visualizes actual vs predicted prices
 🧪 Calculates MSE, MAE, R², Adjusted R²
